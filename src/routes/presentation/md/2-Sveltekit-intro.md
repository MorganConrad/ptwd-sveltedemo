## Sveltekit - a Fremework for building web application in Svelte

 [Generally works with web standards](https://kit.svelte.dev/docs/web-standards)

 ### More like Next

 - filesystem-based routing
 - SSR (or CSR)
 - data loading
 - scoped CSS
 - page-specific rendering modes
 - offline support
